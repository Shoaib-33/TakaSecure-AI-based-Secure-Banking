# Local model artifacts

Place the downloaded fine-tuned LoRA archive and checksum here:

- `takasecure-adapter-v3.zip`
- `takasecure-adapter-v3.zip.sha256`

Large archives are intentionally excluded from Git, while the small SHA-256
checksum is versioned. Publish adapters through a model registry or GitHub
Release and keep reproducible training code, metrics, and documentation in the
source repository.
